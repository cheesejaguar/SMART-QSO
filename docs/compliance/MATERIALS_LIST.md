## 1. Purpose

This document provides a comprehensive list of materials used in the SMART-QSO CubeSat with outgassing compliance status per ASTM E595.

## 2. Outgassing Requirements

### 2.1 Criteria

Per NASA outgassing requirements:
- **TML (Total Mass Loss):** < 1.0%
- **CVCM (Collected Volatile Condensable Materials):** < 0.10%

### 2.2 Reference Database

NASA Outgassing Database: https://outgassing.nasa.gov/

## 16. Prohibited Materials

The following materials are NOT used in SMART-QSO:

| Material | Reason |
|----------|--------|
| Cadmium | Toxic, outgassing |
| Mercury | Toxic |
| Zinc (pure) | Whisker growth, sublimation |
| PVC | Outgassing, chlorine release |
| Silicone rubber (uncured) | High outgassing |
| RTV silicone | High CVCM |

## 17. Materials Requiring Special Handling

| Material | Concern | Mitigation |
|----------|---------|------------|
| Li-ion battery | Flammability | Proper charging, thermal protection |
| Nichrome wire | High temp during deployment | Thermal isolation |
| Adhesives | Cure time | Full cure before TVAC |
